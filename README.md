This repository contains the source files, figures, and supplementary data for the preprint:

**"Maximizing Δ_topo for Fault-Tolerant Topological Qubits"**

## 📂 Contents

- `main.tex` – REVTeX manuscript
- `main.bib` – Bibliography (polished, arXiv-safe)
- `supplementary.tex` – Supplementary Information wrapper
- `figures/` – Final PDF figures
- `src/` – Python scripts and Colab notebook for reproducibility
- `supplementary/` – Data (CSV, workflow diagrams)
- `.github/workflows/` – CI for bibliography linting and auto-fix

## 🔬 Figures

All figures are reproducible via Python/Matplotlib scripts:

- **Fig. 1** – Conceptual topological qubit schematic (`src/fig1_qubit.py`)
- **Fig. 2** – Energy gap barrier plot (`src/fig2_gap_barrier.py`)
- **Fig. 3** – Excitonic material blueprint schematic (`src/fig3_materials_blueprint.py`)
- **Fig. 4** – Δ_topo scaling laws (`src/fig4_scaling.py`)
- **Fig. 5** – Benchmark roadmap (gap vs. overhead)
- **Fig. 6** – Milestone timeline

## 📊 Supplementary Data

- `supplementary_table1.csv` – Δ_topo estimates vs. B, ε, κ
- `supplementary_fig1_workflow.pdf` – Workflow schematic
